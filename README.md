# API_IEB
Métodos para cliente y supermercado

Conexión  a la BD 
1) Configurar BD con estos datos. 
    host: 'localhost',
    user: 'root',
    password: 'dev1234',
    database: 'ieb',
    port: 3306
 2) Conectar servidor MySQL
 3) Script_InvertirEnBolsa_V1
 4) En el proyecto api npm install
 5) node src/app
 6) Dirección de API: localhost:3001
